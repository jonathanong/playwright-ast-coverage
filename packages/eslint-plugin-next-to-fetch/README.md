# eslint-plugin-next-to-fetch

ESLint and Oxlint rules that keep `fetch()` calls statically analyzable by
`next-to-fetch`.

```sh
npm install --save-dev eslint-plugin-next-to-fetch
```

The plugin exports `configs.recommended` with:

- `next-to-fetch/static-fetch-url`
- `next-to-fetch/static-fetch-method`

See [ESLint and Oxlint plugins](../../docs/eslint-plugin.md).
